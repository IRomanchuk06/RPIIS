В данная лабораторная работа предназначена для ознакомления с командной строкой операционных систем Windows, MacOS,Linux. В рамках ЛР1 необходимо создать исполненный файл .sh и .bat в соответствии с условием.

Алгоритм выполнения:
1. Делаем запрос на ввод переменной i
2. Определяем четность/нечетность путем деления на двойку и сравнивания получившегося выражения с нулем
3. Если число четное, заявляем командой echo об этом
4. Далее создаем папки (количество папок равно числу) циклично вложенные друг в друга
5. Создаем и вкладываем файл с текущим временем в последнюю папку в каждой начальной папке
6. Если число нечетное создаем рандомные числа (количество=числу) в папку numbers
