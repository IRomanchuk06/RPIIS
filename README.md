# Описание алгоритма, пример запуска и выполнения программы (.bat)
![image](https://github.com/iis-32170x/RPIIS/assets/148707516/8b1ae3ee-ebb6-419f-9dd1-2eeb5983d8dd)
## Алгоритм программы
1. Отключиаем вывод команд на экран.
2. Задаем переменную, которая будет являться абсолютным путем к файлу.
3. Проверяем, существует ли папка, в которой храниться файл. Если нет, то выводим сообщение "Papka dont found" и программа далее не продолжает работу.
4. Задаем переменную количества с начальным значением 0.
5. В переменной директории проверяем наличие файлов, и после переменной количества добавляется +1 к значению за каждый найденный файл.
6. Задаем переменную результата, значение которой будет являться умножением количества найденных файлов и количеством запущенных процессов.
7. Разделяем переменную текущего времени на ЧАСЫ,МИНУТЫ,СЕКУНДЫ. Убирая разделители ": и ," между ними.
8. Задаем переменную названия файла (fileName) значение которой будет являться текущее время, но в обратном порядке (СЕК,МИН,ЧАС)
9. Выводим значение переменной результата и записываем его в абсолютный путь папки, в файл с названием переменной "fileName" и расширением .txt.
### Переменные
- directory: Содержит абсолютный путь к папке.
- count: Содержит значение количества найденных файлов в абсолютном пути к папке.
- result: Содержит значение результата умножения переменной count на количество процессов (NUMBER_OF_PROCESSORS)
- fileName: Содержит значение C_B_A - Сек_Мин_Час
 ## Разбор кода
1. @echo off: Эта команда отключает вывод команд на экран. То есть, при выполнении скрипта команды не будут отображаться в командной строке.
2. set "directory=D:\PZH\pivas": Здесь устанавливается значение переменной directory равным D:\PZH\pivas. Это абсолютный путь к папке.
3. if not exist "%directory%": Проверяем существует ли указанный путь.
4. Если указанный путь не существует, то командой echo Papka dont found, выводим на экран сообщение "Papka dont found".
5. pause: Эта команда останавливает выполнение скрипта и ожидает нажатия клавиши перед продолжением, также таким образом мы увидим в консоли запись из п.4.
6. exit /b: Эта команда завершает выполнение скрипта, если указанный путь все-таки не существует.
7. set count=0: Здесь устанавливается значение переменной count равным 0. Эта переменная будет использоваться для подсчета количества файлов в пути.
8. for /r "%directory%" %%F in (*) do (...): С помощью цикла for и флага /r, рекурсивно перебираем все файлы в указанной папке и подпапках.
9. set /a "count+=1": После нахождения каждого файла с помощью строки из п.8 к значению переменной count прибавляется 1.
10. set /a "result=count * %NUMBER_OF_PROCESSORS%": Здесь вычисляется значение переменной result путем умножения значения переменной count на значение переменной NUMBER_OF_PROCESSORS. Переменная NUMBER_OF_PROCESSORS содержит количество процессоров в системе.
11. for /f "tokens=1-3 delims=:," %%A in ("%TIME%") do (...): Это цикл for с параметром /f, который разбивает значение переменной окружения TIME на токены, используя символы : и , в качестве разделителей. Переменные %%A, %%B и %%C получают значения соответствующих токенов.
12. set "fileName=%%C_%%B_%%A": Здесь мы задаем значение переменной fileName соединяя значения переменных %%C, %%B и %%A в %%C_%%B_%%A. В результате получается значение "секунды_минуты_часы", которая будет являться именем файла.
13. echo %result% > "%directory%\%fileName%.txt": Эта команда записывает значение переменной result в файл с расширением .txt, имя которого формируется из значения переменной fileName, в указанном каталоге. Знак > используется для переноса значения переменной "result" в файл "filename.txt".

 # Описание лагоритма, пример запуска и выполнения программы (.sh)
 
 ![image](https://github.com/iis-32170x/RPIIS/assets/148707516/4bfb2b37-ad2a-4117-9da7-25c8c6765bdb)

 ## Алгоритм программы
 1. Устанавливается переменная directory с путем к каталогу "D:/PZH/pivas".
 2. Проверяется, существует ли указанный каталог. Если каталог не существует, выводится сообщение "Papka not found", и программа завершается.
 3. Устанавливается переменная count со значением 0.
 4. Ищутся все файлы внутри каталога directory и его подкаталогов.
 5. При каждом найденном файле значение переменной count увеличивается на 1.
 6. Вычисляется результат умножения значения переменной count на количество доступных процессоров.
 7. Задается переменная fileName, которая содержит текущее время в формате "HH_MM_SS".
 8. Значение переменной result записывается в текстовый файл с именем $fileName.txt внутри каталога directory.
 ### Переменные
 - directory: Содержит абсолютный путь к папке.
 - count: Содержит значение количества найденных файлов в абсолютном пути к папке.
 - result: Содержит значение результата умножения переменной count на количество процессов (nproc).
 ## Разбор кода
 1. Задается переменная directory со значением пути к каталогу "D:/PZH/pivas".
 2. if if [ ! -d "$directory" ]]; then: Это выражение которое проверяет, является ли "$directory" несуществующим каталогом (-d проверяет, является ли путь каталогом, а ! отрицает результат проверки).
 3. echo "Papka not found" - Эта команда выводит сообщение "Papka not found", если из п.2 было определено, является ли "$directory$" несуществующим каталогом.
 4. exit 1 - Эта команда завершает выполнение программы с кодом 1, что указывает на ошибку в выполнении программы.
 5. while read -r -d '' file; do: Эта строка начинает цикл while и считывает строки из ввода и сохраняет их в переменную file. Опции -r сохраняют строки без интерпретации экранированных символов, а -d '' указывает, что строки разделяются пустыми символами.
 6. ((count++)) - После того, как будет найден каждый файл значение переменной будет увеличиваться на 1. done - Это конец цикла while.
 7. < <(find "$directory" -type f -print0): Данная команда выполняется, чтобы найти все файлы в указанном каталоге "$directory" и его подкаталогах. -type f указывает, что нужны только файлы, а -print0 выводит результаты в формате с нулевыми символами в качестве разделителей. Все найденные файлы передаются в цикл while в качестве ввода.
 8. result=$((count * $(nproc))): Это выражение, которое умножает значение переменной count на количество доступных процессоров и результат записывает в переменную result.
 9. fileName=$(date +"%S_%M_%H"): Задаем значение переменной fileName, команда date, которая задает текущее время, с помощью  +"%S_%M_%H" указываем формат вывода времени в секундах_минутах_часах.
 10. echo "$result" > "$directory/$fileName.txt": Выводим значение переменной "$result$" и перенаправляем ее в файл с названием "filename".
