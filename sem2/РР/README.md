# Алгоритм выполнения работы на ественном языке:
Суть алгоритма: сначала при помощи поиска в глубину(dfs) находятся все циклы графа, а после этого среди них выбираются только подходящие - с определенной длиной, указаной пользователем:  
1) Создаем массив посещенных вершин.
2) Создаем массив всех циклов.
3) Создаем массив пути.
4) i = 1.
5) Берем i-ую вершину за начальную.
6) Добавляем текущую вершину в массив посещенных вершин.
7) Добавляем текущую вершину в массив пути.
8) Рассматриваем соседние вершины текущей:  
    8.1. Если соседней вершины нет в массиве посещеннвых, переходим к пункту 6.  
    8.2. Если соседняя вершина есть в массиве посещенных и не является предком предыдущей:  
        i. Создаем массив цикла.  
        8.2.2) Добавляем в массив цикла элементы(вершины) массива пути и текущую вершину.  
        8.2.3) Добавляем в массив всех циклов элементы(вершины) массива цикла.  
9) Удаляем
10) в
11) Увеличиваем i на 1.
12) Если i <= n(n - количество вершин графа), переходим к пункту 5.
13) j = 1.
14) Считаем длину j-ого цикла массива всех циклов(сумма весов ребер, принадлежащих циклу).
15) Если длина j-ого цикла массива всех циклов равна длине, указанной пользователем, выводим цикл.
16) Если j <= m(m - количество циклов в массиве всех циклов), переходим к пункту 14.
