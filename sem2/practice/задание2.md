# Практическое задание 2, вариант 2(gwf-файл)

## Условие

## `(a + b)^3 = a^3 + b^3 + 3(a^2)b + 3(b^2)a`

## Пояснение

Для вывода формул потребуется переменные `a` & `b`.

Для того, чтобы вывести формулу нужно:

- получить `(a + b)`
- получить `(a + b)^3`
- получить `b^3`
- получить `a^3`
- получить `b^2`
- получить `a^2`
- получить `3*b`
- получить `3*a`
- получить `3*a*b^2`
- получить `3*b*a^2`
- получить `a^3 + b^3 + 3(a^2)b + 3(b^2)a`
- связать `(a + b)^3` и `a^3 + b^3 + 3(a^2)b + 3(b^2)a`
